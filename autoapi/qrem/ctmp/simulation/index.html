qrem.ctmp.simulation

simulation.py: Quantum State Simulation under CTMP Model

Part of the CTMP subpackage, this module simulates the evolution of quantum states under the influence of quantum errors as modeled by the CTMP model. It uses Gillespie’s algorithm to model state transitions over time.

Functions

generate_sample(CTMP_model: CTMPModelData, input_state: str, t: float=1) -> str

Generates a quantum state sample by simulating state transitions based on the CTMP model. It applies Gillespie’s algorithm to model these transitions over a specified time period.
