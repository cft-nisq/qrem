.. py:function:: divide_qubits_in_the_clusters(qubits_indices: List, clusters_specification: List[List]) -> List[tuple]

   A function used to divide qubits into clusters

   :param qubits_indices: a list of qubits indices defined upon .....
   :type qubits_indices: list
   :param clusters_specification:
                                  - a nested list containing description of clusters to be created:

                                      structure of the list [[element_1, element_2], ...]:
                                          element_1 - integer encoding locality of a cluster
                                          element_2 - number of clusters of given locality

                                      e.g. [[3,1],[2,2]] encodes 1 3 qubits cluster and two 2 qubits clusters
   :type clusters_specification: list

   :returns: list with qubits assigned to clusters
   :rtype: cluster_list
